Overview & Architecture

🔒 Diamond Cartel Exclusive

This script is exclusively for members of Diamond Cartel (ID: 48822). The backend and scripts verify your faction membership via the Torn API on every session. If you leave the faction the tool stops working immediately. Developed by OfficialSVK.

War Intel is a full-stack war intelligence tool combining browser userscripts with a shared backend server. It automatically detects your active ranked war, tracks enemy member statuses in real time, logs flight transitions, and serves combined data to all faction members simultaneously.

Diamond Cartel Members (Desktop + TornPDA) │ │ /api/auth · /api/war-state · /api/observations ▼ ┌─────────────────────────────────────────────┐ │ FastAPI Backend (Railway) │ │ • War detection every 2 minutes │ │ • Enemy faction poll every 30 seconds │ │ • FFScouter cache refresh every 5 minutes │ │ • Contributor key registry │ └──────────────┬──────────────────────────────┘ │ ┌──────────────▼──────────────┐ ┌──────────────────────┐ │ PostgreSQL Database │ │ Torn API │ │ members · flights │ │ + FFScouter API │ │ auth_cache · contributor_ │ │ │ │ keys · observations │ │ │ └─────────────────────────────┘ └──────────────────────┘

When the backend is reachable (green dot in status bar) scripts make a single call to /api/war-state instead of three separate Torn API calls — reducing your personal API quota usage and providing shared flight history that builds up over time as more members use the tool.

API Keys You Need

1. Your Personal Torn API Key

Used to authenticate you with the backend and as a fallback if the backend is unreachable.

🔑 Generate your Torn API Key

Direct link (pre-fills required permissions):
Generate key → faction: basic, members, rankedwars

PermissionRequired?Why
faction → basicYesIdentifies your faction and enemy member statuses
faction → rankedwarsYesDetects your active ranked war and scores
faction → membersYesFetches the enemy faction member list
⚠ Note: A Limited Access or Full Access key includes all of the above by default. Custom keys must have all three faction selections ticked.

2. FFScouter Key (Optional but Recommended)

Provides estimated battle stats and Fair Fight scores for enemy members. Without it the Est. Stats and FF columns will be empty.

🔑 Register with FFScouter

Visit ffscouter.com and sign up using your Torn API key. You must read and agree to their Data Policy — the script will prompt you to confirm this.

Your FFScouter key is the same 16-character Torn API key you registered with on their site.

Contributor Keys — Improving Coverage

The backend server polls the enemy faction every 30 seconds using a single dedicated key. You can improve flight detection accuracy by registering a Contributor Key — a limited Torn API key that the backend registers and validates, increasing its data coverage network.

✓ Why this helps: The more members who contribute keys, the denser the coverage network. Multiple contributors catching the same flight transition from different refresh cycles means more accurate departure timestamps and better estimated arrival times.

How to generate and register a Contributor Key

1
Generate a limited key on Torn

Click this link — it pre-fills the exact permissions needed:
⚙ Generate Contributor Key on Torn →

2
Copy the generated key

Torn creates the key automatically with only faction → basic, members, rankedwars permissions. Copy it.

3
Paste into script settings

Open ⚙ Settings in War Intel and paste the key into the Contributor Key field. Click Save & Fetch — the backend registers it automatically.

⚠ Security note: The contributor key only has read access to faction data — it cannot perform any in-game actions. You can delete it from Torn's API settings at any time to immediately revoke it. The backend stores only a SHA-256 hash of the key — never the raw key itself.

⚡ War Mode

Found in ⚙ Settings — a red toggle switch labelled ⚡ War Mode.

  • Drops the refresh interval from 60s → 30s (desktop) and 90s → 30s (TornPDA)
  • Doubles the frequency of flight transition detection, hospital release tracking, and status updates
  • A ⚡ WAR MODE badge appears in the status bar so you always know it's active
  • Setting is saved and persists between page changes
  • Uses approximately 4 API calls per minute — well within Torn's 100/min limit even with War Mode on
✓ Tip: With 10 faction members running War Mode simultaneously, the collective detection rate reaches approximately one check every 3 seconds — dramatically improving flight tracking accuracy.

Flight Tracking

The backend logs flight transitions every time it detects an enemy member changing from any status to "Traveling". The script's flight view shows:

  • Route — e.g. TRN → JAP or HAW → TRN for return flights
  • Arrival countdown — exact if Torn's API provides an until timestamp; otherwise estimated from known standard airline flight times with a ~ prefix
  • Flights logged — how many flights we've recorded for that player this war

Troubleshooting

Orange dot — "fallback mode"

The backend is temporarily unreachable. The script falls back to direct Torn API calls and continues working normally. The backend usually recovers within a minute. Check Railway logs if it persists.

"No active ranked war found"

Normal outside of war periods. The dashboard detects wars automatically — no action needed.

"Access Denied" screen

The script verified your Torn API key and found you are not a member of Diamond Cartel (ID 48822). Make sure you are using your own key and that you are still in the faction.

code 23 — "only available in API v2"

Your API key is missing a required permission. Use the direct link in API Keys to generate a new one with the correct permissions.

FFScouter shows "—" in Est. Stats / FF

  • FFScouter only has estimates for players who have been attacked recently by other FFScouter users.
  • Confirm your FFScouter key is registered at ffscouter.com.
  • The backend refreshes the FFScouter cache every 5 minutes — wait a moment after war detection.

Flight arrivals show "~" prefix

This is expected — ~ means the arrival time is estimated from the standard airline flight time table rather than provided by Torn's API. Players with Airstrip or Business Class perks will land earlier than shown.

Export does nothing on iOS

iOS blocks file downloads from in-app browsers. The export copies the report to your clipboard instead — a green toast confirms this. Open Torn mail and paste.

PDA panel disappears when hidden

Look for the small red ⚔ WAR tab on the left or right edge of your screen (depending on your Settings preference) and tap it to restore the panel.

Script not working after leaving the faction

By design — the script verifies faction membership on every session. This is intentional security behaviour.

General errors

Re-install from Greasy Fork using the links at the top of this guide. If the backend is causing issues, check the Railway deployment logs.
