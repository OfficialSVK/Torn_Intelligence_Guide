Overview

War Intel is a userscript that automatically detects your active Torn ranked war, pulls enemy faction member data, and enriches it with battle stat and Fair Fight predictions from FFScouter — displayed in a live dashboard that refreshes automatically.

  • Desktop — Floating, resizable panel with sortable columns, filter views, attack buttons, hospital timers, and a formatted HTML export for Torn mail. Installed via Tampermonkey.
  • TornPDA (Mobile) — Full-width bottom sheet with filter tabs, card-based member view, FF scores, hospital/jail timers, travel status, and one-tap attack links. Installed via TornPDA's built-in script manager.

What You Need — API Keys

1. Torn API Key

Your Torn API key lets the script fetch faction war data and enemy member information.

🔑 Generating your Torn API Key

Settings page: torn.com → Preferences → API

Direct link (pre-fills required permissions):
Generate key with faction: basic, rankedwars, members

PermissionRequired?Why it's needed
faction → basicYesIdentifies your faction name and ID
faction → rankedwarsYesDetects your active ranked war and scores
faction → membersYesFetches the enemy member list with live status
⚠ Note: A Limited Access or Full Access key includes all of the above by default. If using a Custom Key, make sure all three faction selections are ticked.

2. FFScouter Key (Optional but Recommended)

FFScouter is a free community tool that estimates enemy battle stats and Fair Fight scores. Without it the dashboard shows names, IDs, levels and status — but Est. Stats, FF scores and stat distribution will be empty.

🔑 Registering with FFScouter

Visit ffscouter.com and sign up using the same Torn API key you use for this script. You must read and agree to the FFScouter Data Policy — the script will prompt you to confirm this when you enter your key.

Your FFScouter key is the same 16-character Torn API key you registered with on their site.

Desktop Installation (Tampermonkey)

Step 1 — Install Tampermonkey

Tampermonkey is a free browser extension that runs userscripts. Install it from tampermonkey.net — supports Chrome, Edge, Firefox, and Safari.

⚠ Chrome users: If the script doesn't appear after installation, enable Developer Mode in Chrome's Extensions settings. See tampermonkey.net/faq#Q209.

Step 2 — Install the Script

1
Install from Greasy Fork

Visit greasyfork.org — Torn War Intelligence Dashboard and click Install this script.

2
Confirm installation

Tampermonkey opens a new tab showing the script details. Click Install to confirm.

3
Navigate to Torn

Open torn.com. The ⚔ WAR INTEL panel appears in the top-right corner of the page.

Step 3 — Enter Your API Keys

1
Open Settings

Click ⚙ Settings in the War Intel panel header.

2
Agree to FFScouter policy

Read the FFScouter data policy (linked in the panel) and tick the checkbox to confirm.

3
Enter your keys

Paste your Torn API key into TORN API KEY and optionally your FFScouter key into FFSCOUTER KEY.

4
Save & Fetch

Click 💾 Save & Fetch. Enemy members load automatically if your faction is in an active ranked war.

TornPDA Installation (Mobile)

TornPDA is a mobile client for Torn with a built-in userscript manager — no separate extension needed.

Step 1 — Install TornPDA

Download TornPDA from the App Store (iOS) or Google Play (Android) and log in with your Torn account.

Step 2 — Install the Script

1
Open TornPDA Userscripts

Tap the profile icon → Settings → Userscripts.

2
Add from Greasy Fork

Tap + and enter the URL: greasyfork.org — Torn War Intelligence TornPDA Mobile

3
Enable and restart

Make sure the script is toggled ON. Fully restart TornPDA.

4
Navigate to Torn

Open any Torn page. The ⚔ WAR INTEL bar appears at the bottom of the screen. Tap it to expand.

Step 3 — Enter Your API Keys

Tap ⚙ in the panel header, agree to the FFScouter policy, enter your keys, and tap 💾 Save & Fetch.

Export (Mobile)

  • iOS — Tap ⬇ to copy the full report HTML to your clipboard. A green toast confirms it. Open Torn mail and paste directly.
  • Android — Tap ⬇ to download a TXT file containing the HTML report.

The report groups members into threat tiers with your faction vs enemy header and clickable Torn profile links — ready to paste into faction mail or a forum post.

Export Report Format

Both desktop and mobile produce the same export format — HTML markup that renders in Torn mail and forum posts. The report includes:

  • Header — Your faction name vs enemy faction name, plus total enemy count
  • Threat tier sections — Members grouped and sorted by estimated stats within each tier
  • Per member — Name (clickable profile link), Torn ID, estimated stats, and stat distribution where available

Troubleshooting

"No active ranked war found"

Normal outside of war periods. The dashboard detects wars automatically — no action needed.

"Torn API: ... (code 23)"

Your API key is missing a required permission. Use the direct link in the What You Need section to generate a new key, or manually add faction → basic, rankedwars, and members to an existing key.

FFScouter shows "—" in Est. Stats or FF columns

  • FFScouter only has estimates for players attacked recently by other FFScouter users. New or inactive players may have no data.
  • Confirm your FFScouter key is registered at ffscouter.com.

Hospital timer shows but doesn't tick

The ticker starts when the page loads. Try refreshing the Torn page. If the timer shows "—" the player's status has no until timestamp — this happens for traveling players or when the API returns a non-hospital/jail state.

No flight arrival timer for traveling players

Torn keeps flight arrival times private — they are not included in the API response. The Timer view shows who is traveling and their destination, but a countdown isn't possible with the available data.

Panel not appearing on Desktop

  • Check the script is enabled in the Tampermonkey dashboard.
  • Chrome users: enable Developer Mode in Chrome Extensions — see tampermonkey.net/faq#Q209.
  • If you previously clicked ✕ to hide the panel, look for the small ⚔ WAR tab on the left edge of the screen and click it to restore the panel.

Script not appearing in TornPDA

  • Fully close and reopen TornPDA after installing the script.
  • Confirm the script is toggled ON in Settings → Userscripts.
  • Make sure you are on a torn.com page — the script only runs on Torn.

Export does nothing on iOS

iOS blocks file downloads from in-app browsers. The export copies the report to your clipboard instead — a green toast confirms this. Open Torn mail and paste.

General errors or broken behaviour

Re-install by uninstalling the current script and reinstalling from Greasy Fork using the links at the top of this guide.
